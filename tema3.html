Aplicatie JavaScript care sa realizeze o procesare de imagini

Descrierea aplicatiei

Acest script JavaScript utilizează API-ul https://dog.ceo/api/breeds/image/random pentru a obține o imagine aleatorie cu câini și o afișează pe un element canvas într-un document HTML. După afișarea imaginii, scriptul continuă prin aplicarea unor transformări, inclusiv afișarea imaginii în oglindă și aplicarea unui algoritm simplu de edge detection (pentru a evidenția marginile imaginii) și afișează rezultatele filtrării pe canvas, cu o întârziere de 1 secundă între fiecare etapă.
